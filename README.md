# 🎨 Color Converter

Convert colors between HEX, RGB, HSL, and CMYK. Includes color picker and live preview.

## 🚀 Features

- **Multi-Format** — HEX, RGB, HSL, CMYK conversion
- **Color Picker** — Visual color selection
- **Live Preview** — See color as you type
- **Copy Values** — One-click copy for each format
- **Contrast Checker** — WCAG contrast ratio
- **No Server** — Runs entirely in browser

## 🎯 Why This?

Color converters are often ad-filled. This one is clean, visual, and local.

## 🛠 Tech Stack

- Next.js 14
- React
- Tailwind CSS
- shadcn/ui

## 📦 Setup

```bash
git clone https://github.com/thehaardik/color-converter.git
cd color-converter
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
